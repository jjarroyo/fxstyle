JTitleBar

Componente que reemplaza la barra de título nativa del sistema operativo, permitiendo un diseño inmersivo y personalizado con soporte automático para modo claro y oscuro.

Características

  • Arrastre de ventana fluido.
  • Doble clic en la barra para maximizar/restaurar.
  • Botones integrados de minimizar, maximizar y cerrar con íconos vectoriales al estilo Windows 11.
  • Animación fluida Scale+Translate al maximizar o restaurar, evitando el flash o estiramiento de contenido de JavaFX por defecto.
  • Soporte nativo para .root.dark.
  • Zona derecha personalizable para añadir perfiles, barras de búsqueda o botones propios antes de los controles de la ventana.

Ejemplo de Uso Rápido

La forma más sencilla de utilizar JTitleBar es usando su factory estático install(), el cual asume el control del Stage, lo configura como TRANSPARENT e inyecta las sombras y bordes redondeados necesarios.

import com.jjarroyo.components.JTitleBar;
import com.jjarroyo.components.JIcon;

@Override
public void start(Stage primaryStage) {
    // 1. Crear el contenido principal
    VBox myAppContent = new VBox(new Label("Hola Mundo"));
    
    // 2. Instanciar y configurar el JTitleBar
    JTitleBar titleBar = new JTitleBar()
        .setTitle("Mi Aplicación JavaFX")
        .setIcon(new JIcon(JIcon.APPS))
        .setOnCloseRequest(() -> {
            System.out.println("Guardando datos antes de salir...");
            Platform.exit();
        });

    // 3. Opcional: Agregar nodos custom (ej. un input de búsqueda) a la derecha
    titleBar.addRightNode(new JSearchInput());

    // 4. Instalar en el Stage (esto creará la Scene y ajustará el Stage)
    JTitleBar.install(primaryStage, titleBar, myAppContent, 1024, 768);

    primaryStage.show();
}

Referencia de API (Fluent Builder)

Método Descripción
setTitle(String title) Establece el texto del título de la ventana.
setIcon(Node icon) Añade un nodo o ícono a la izquierda del texto del título.
addRightNode(Node node) Agrega un nodo en el espacio a la derecha, justo antes de los botones de la ventana.
setOnCloseRequest(Runnable action) Acción a realizar al pulsar el botón "X". Por defecto hace Platform.exit().
setAnimated(boolean value) Habilita (true) o deshabilita (false) la animación de maximizado y minimizado.
JTitleBar.install(...) (static) Configura el Stage (Style TRANSPARENT), asocia el título, el contenido y la escena.

Detalles de Implementación

A diferencia de las barras de título custom tradicionales en JavaFX que sufren durante el redimensionamiento, JTitleBar utiliza un enfoque de Scale + Translate.

Al maximizar la ventana, el Stage toma el tamaño del sistema instantáneamente permitiendo un layout a máxima resolución. Visualmente, el contenedor principal de la aplicación es escalado hacia abajo para simular su tamaño previo, y luego animado con un ScaleTransition hacia un scale de 1.0 usando una curva de tipo cubic-ease-out.

Esto asegura que todo vector y texto se vea nítido durante toda la transición, replicando la sensación de las aplicaciones nativas en Windows.
