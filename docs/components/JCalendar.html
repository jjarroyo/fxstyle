// Uso básico
JCalendar calendar = new JCalendar();

// Escuchar selección
calendar.setOnDateSelected(date -> {
    System.out.println("Fecha: " + date);
});

// Resaltar fechas de eventos
calendar.setHighlightedDates(Set.of(
    LocalDate.now(),
    LocalDate.of(2026, 4, 15)
));

Características Principales

Navegación Fluida

Botones integrados para cambiar de mes y botón "Hoy" para retorno rápido.

Gestión de Eventos

Capacidad para resaltar múltiples fechas de forma visual (ej: citas, tareas).

Totalmente Observable

Propiedad selectedDate para binding bidireccional y reactividad.

Personalización CSS

Control total sobre el aspecto de cada día (hoy, seleccionado, otros meses, fines de semana).

Clases CSS

Clase Descripción
j-calendar Contenedor raíz del componente.
j-calendar-day-today Estilo para el día actual.
j-calendar-day-selected Estilo para el día seleccionado por el usuario.
j-calendar-day-highlighted Estilo para fechas marcadas como eventos.
j-calendar-day-weekend Estilo específico para sábados y domingos.

API Reference

Método Descripción
selectedDateProperty() Retorna la propiedad observable de la fecha seleccionada.
setOnDateSelected(Consumer<LocalDate>) Define una acción a ejecutar cuando se hace clic en una fecha.
setHighlightedDates(Set<LocalDate>) Establece el conjunto de fechas a resaltar visualmente.
setSelectedDate(LocalDate) Cambia la fecha seleccionada y navega al mes correspondiente.
