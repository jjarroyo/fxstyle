JFloatingButton

Un Floating Action Button (FAB) que se posiciona automáticamente dentro de su contenedor padre. Soporta múltiples posiciones (ej. abajo-derecha, centro, arriba-izquierda) y estilos adaptables (circular redondo o píldora extendida con texto e ícono).

Ejemplo de Uso

Vista Previa: JFloatingButton en Bottom-Right (Imagina el botón flotando aquí abajo) 
// 1. Botón FAB básico redondo (Abajo Derecha)
JFloatingButton fabBasic = new JFloatingButton(JIcon.ADD);
fabBasic.setPosition(JFloatingButton.Position.BOTTOM_RIGHT);
fabBasic.addClass("btn-primary");

// 2. Botón FAB extendido con texto (Arriba Izquierda)
JFloatingButton fabExtended = new JFloatingButton("Crear Tarea", JIcon.EDIT);
fabExtended.setPosition(JFloatingButton.Position.TOP_LEFT);
fabExtended.addClass("btn-success");

// Añadir a un Pane (AnchorPane, StackPane, Region...)
StackPane container = new StackPane();
container.getChildren().addAll(fabBasic, fabExtended);

Posiciones Disponibles

El botón escucha al contenedor padre y se ajusta dinámicamente usando JFloatingButton.Position:

  • TOP_LEFT
  • TOP_CENTER
  • TOP_RIGHT
  • CENTER_LEFT
  • CENTER
  • CENTER_RIGHT
  • BOTTOM_LEFT
  • BOTTOM_CENTER
  • BOTTOM_RIGHT

API Reference

Método Descripción
JFloatingButton() Constructor básico vacío.
JFloatingButton(JIcon) Crea un botón circular sólo con icono.
JFloatingButton(String, JIcon) Crea un botón extendido (Píldora) con texto e icono.
setPosition(Position) Define la posición del botón respecto a los bordes de su contenedor.
setMargin(double) Establece el margen (espaciado) de separación con los bordes (por defecto 20px).
setExtended(boolean) Habilita u oculta el formato extendido del FAB de ser circular a píldora.
