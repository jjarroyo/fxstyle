JNotification

Un componente de notificaciones avanzado que incluye un disparador con badge dinámico y un panel desplegable detallado para una experiencia de usuario premium.

Vista Previa

3
Disparador (Trigger)
Notificaciones
A

Actualización Disponible

Hace 5m

Hay una nueva versión de FxStyle lista para descargar.

S

Venta Realizada

Hace 1h

Has recibido un pago de $500.00 de un cliente nuevo.

Uso Básico

// 1. Instanciar el componente
JNotification jNotification = new JNotification();

// 2. Crear y añadir notificaciones
JNotificationItem item = new JNotificationItem(
    "Actualización Disponible",
    "Hay una nueva versión de FxStyle disponible para descargar.",
    "/path/to/icon.png",
    LocalDateTime.now().minusMinutes(5)
);

jNotification.addNotification(item);

// 3. Integrar en tu layout (ej. Header)
jHeader.setRight(jNotification);

Características

Badge Dinámico

El badge contador se actualiza automáticamente al añadir notificaciones y desaparece cuando el conteo es cero.

Tiempo Relativo

Formatea automáticamente las fechas a texto legible como "Hace 2m", "Ayer", etc.

Scrollable Body

El panel principal incluye un ScrollPane estilizado para manejar largas listas de notificaciones sin afectar el layout.

Clean Overlay

Utiliza JPopover internamente para un despliegue con animaciones de Fade y Scale.

API Reference

Método Descripción Parámetros
addNotification(JNotificationItem item) Añade una nueva notificación al inicio de la lista. JNotificationItem
clearNotifications() Elimina todas las notificaciones de la lista. Ninguno
markAllAsRead() Cambia el estado de todas las notificaciones a leídas. Ninguno
setOnViewAll(Runnable callback) Establece la acción para el botón "Ver todas". Runnable
