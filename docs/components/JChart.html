JChart

Componente de gráficos animados y profesionales para JavaFX. Soporta barras, líneas, pie, donut, área, dispersión y barras horizontales con una API ultra-simple.

Canvas-based Animated 7 tipos 5 paletas Fluent API

🚀 Inicio Rápido

Crear un gráfico es tan simple como una sola línea de código:

// Gráfico de barras
JChart chart = JChart.bar("Ventas Mensuales",
    new String[]{"Ene", "Feb", "Mar", "Abr"},
    new double[]{120, 200, 150, 250});

// Agregarlo a cualquier layout
myContainer.getChildren().add(chart);

🎨 Paletas de Colores

5 paletas profesionales incluidas. Usa .withPalette() para cambiar los colores.

PALETTE_VIBRANT (default)

PALETTE_OCEAN

PALETTE_SUNSET

PALETTE_FOREST

PALETTE_SLATE

// Usar paleta predefinida
chart.withPalette(JChart.PALETTE_OCEAN);
chart.withPalette(JChart.PALETTE_SUNSET);
chart.withPalette(JChart.PALETTE_FOREST);
chart.withPalette(JChart.PALETTE_SLATE);

// Paleta personalizada
chart.withPalette(new Color[]{
    Color.web("#8b5cf6"),
    Color.web("#ec4899"),
    Color.web("#06b6d4")
});

⚙️ Personalización

// Tamaño personalizado
chart.withSize(700, 350);

// Mostrar/ocultar leyenda
chart.withLegend(false);

// Mostrar/ocultar valores sobre las barras
chart.withValues(false);

// Mostrar/ocultar grid
chart.withGrid(false);

// Deshabilitar animación
chart.withAnimation(false);

// Actualizar datos dinámicamente
chart.setData(newLabels, newValues);
chart.setData(newLabels, newMultiValues, newSeriesNames);

🎯 Ejemplo: Dashboard Completo

// Dashboard con múltiples gráficos
HBox row1 = new HBox(20);

// Donut de distribución
JChart cropDonut = JChart.donut("Cultivos",
    new String[]{"Café", "Caña", "Maíz"},
    new double[]{35, 25, 20});
cropDonut.withSize(280, 240);
cropDonut.withPalette(JChart.PALETTE_FOREST);

// Barras agrupadas de cosecha
JChart harvestBars = JChart.bar("Cosecha (ton)",
    new String[]{"Q1", "Q2", "Q3", "Q4"},
    new double[][]{
        {220, 340, 280, 150},
        {190, 310, 320, 180}
    },
    new String[]{"2024", "2025"});

row1.getChildren().addAll(cropDonut, harvestBars);

// Línea de precipitación
JChart rainLine = JChart.line("Lluvia (mm)",
    months, rainData);
rainLine.withPalette(JChart.PALETTE_OCEAN);

// Área de crecimiento
JChart growthArea = JChart.area("Crecimiento (cm)",
    weeks, growthData);
growthArea.withPalette(JChart.PALETTE_FOREST);

📖 API Reference

Métodos de Fábrica (estáticos)

Método Descripción
JChart.bar(title, labels, values) Gráfico de barras vertical.
JChart.bar(title, labels, multiValues, series) Barras agrupadas con múltiples series.
JChart.horizontalBar(title, labels, values) Barras horizontales.
JChart.line(title, labels, values) Gráfico de línea.
JChart.line(title, labels, multiValues, series) Multi-línea.
JChart.pie(title, labels, values) Gráfico circular.
JChart.donut(title, labels, values) Gráfico de dona con total en el centro.
JChart.area(title, labels, values) Gráfico de área con gradiente.
JChart.area(title, labels, multiValues, series) Multi-área superpuesta.
JChart.scatter(title, labels, values) Gráfico de dispersión.
JChart.scatter(title, labels, multiValues, series) Dispersión multi-serie.

Métodos de Configuración

Método Descripción
withPalette(Color[]) Cambia la paleta de colores.
withSize(int w, int h) Tamaño del canvas en píxeles.
withLegend(boolean) Mostrar/ocultar leyenda.
withValues(boolean) Mostrar/ocultar valores sobre datos.
withGrid(boolean) Mostrar/ocultar líneas de grid.
withAnimation(boolean) Habilitar/deshabilitar animación.
setData(labels, values) Actualiza los datos y re-anima.
refresh() Fuerza re-dibujo con animación.

Constantes de Paleta

Constante Estilo Uso Recomendado
PALETTE_VIBRANT General, dashboards coloridos.
PALETTE_OCEAN Corporativo, reportes serios.
PALETTE_SUNSET Marketing, datos de alertas.
PALETTE_FOREST Agrícola, ambiental, salud.
PALETTE_SLATE Minimalista, impresión B/N.
