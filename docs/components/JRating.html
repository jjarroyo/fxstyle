JRating

Componente interactivo de calificación con estrellas. Soporta múltiples tamaños, variantes de color, modo solo lectura, medias estrellas y binding de propiedades.

Uso Básico

Vista previa interactiva (HTML simulado):

★ ★ ★ ★ ★ 3.0 / 5.0 
// Rating básico de 5 estrellas
JRating rating = new JRating();

// Rating con valor inicial
JRating rating = new JRating(5, 3);

// Escuchar cambios
rating.setOnRatingChanged(value -> {
    System.out.println("Rating: " + value);
});

API Reference

Método Descripción
JRating() Constructor por defecto: 5 estrellas, valor 0.
JRating(int maxStars) Constructor con número personalizado de estrellas.
JRating(int maxStars, double initialRating) Constructor con estrellas y valor inicial.
setRating(double) Establece el valor del rating programáticamente.
getRating() Obtiene el valor actual del rating.
ratingProperty() Propiedad observable para binding bidireccional.
setMaxStars(int) Cambia el número máximo de estrellas.
setReadOnly(boolean) Activa/desactiva el modo solo lectura.
setAllowHalf(boolean) Habilita calificaciones en incrementos de 0.5.
setOnRatingChanged(Consumer<Double>) Callback que se ejecuta al cambiar el rating.
withSize(String) Tamaño: "rating-sm", "rating-md", "rating-lg".
withColor(String) Color: "rating-primary", "rating-danger", "rating-success", "rating-info", "rating-dark".
withStyle(String...) Agrega una o más clases CSS personalizadas.

Clases CSS

Clase Descripción
.j-rating Contenedor principal del componente.
.j-rating-star Cada estrella individual.
.rating-sm / .rating-lg Variantes de tamaño.
.rating-primary, .rating-danger, etc. Variantes de color.
.rating-readonly Modo solo lectura (aplicado automáticamente).
