JConfirmDialog

Diálogo de confirmación reutilizable con variantes de tipo (INFO, SUCCESS, WARNING, DANGER), iconos automáticos, callbacks síncronos/asíncronos y loading state.

Uso Básico (una línea)

La forma más simple: un título y un callback.

// Confirmación simple
JConfirmDialog.show("¿Estás seguro?", () -> {
    System.out.println("Confirmado!");
});

Con Mensaje Descriptivo

Agrega un mensaje de contexto debajo del título.

JConfirmDialog.show(
    "¿Guardar cambios?",
    "Los datos modificados se guardarán permanentemente.",
    () -> guardarDatos()
);

Variantes de Tipo

Cada tipo tiene un icono y color automático.

Type.INFO

Icono: ℹ️ Info — Botón azul

Type.SUCCESS

Icono: ✓ Check — Botón verde

Type.WARNING

Icono: ⚠ Warning — Botón amarillo

Type.DANGER

Icono: 🗑 Delete — Botón rojo

// Con tipo DANGER
JConfirmDialog.show(
    "¿Eliminar registro?",
    "Esta acción no se puede deshacer.",
    JConfirmDialog.Type.DANGER,
    () -> eliminarRegistro()
);

Shortcut: showDelete

Método estático pre-configurado para eliminaciones (DANGER + icono de papelera).

JConfirmDialog.showDelete(
    "¿Eliminar el usuario 'Juan'?",
    () -> deleteUser(userId)
);

Fluent API (Personalización completa)

Para control total sobre cada aspecto del diálogo.

new JConfirmDialog()
    .setTitle("Enviar reporte")
    .setMessage("El reporte mensual se enviará a gerencia@empresa.com")
    .setType(JConfirmDialog.Type.SUCCESS)
    .setIcon(JIcon.EMAIL)            // Icono personalizado
    .setConfirmText("Enviar ahora")  // Texto del botón confirmar
    .setCancelText("Más tarde")      // Texto del botón cancelar
    .setOnConfirm(() -> enviarReporte())
    .setOnCancel(() -> System.out.println("Cancelado"))
    .show();

Acción Asíncrona (Loading State)

El botón muestra un spinner mientras la acción se ejecuta en background.

new JConfirmDialog()
    .setTitle("¿Eliminar usuario?")
    .setType(JConfirmDialog.Type.DANGER)
    .setConfirmText("Eliminar")
    .setOnConfirmAsync(
        () -> userService.delete(userId),   // Se ejecuta en hilo background
        success -> {                         // Callback en hilo JavaFX
            if (success) refreshTable();
        }
    )
    .show();

API Reference

Constructores

Constructor Descripción
JConfirmDialog() Crea un diálogo vacío con título por defecto.
JConfirmDialog(String title) Crea un diálogo con título.
JConfirmDialog(String title, String message) Crea un diálogo con título y mensaje.

Métodos estáticos

Método Descripción
show(title, onConfirm) Confirmación básica.
show(title, message, onConfirm) Con mensaje descriptivo.
show(title, message, type, onConfirm) Con tipo de variante.
showDelete(message, onConfirm) Pre-configurado para eliminar (DANGER).

Métodos de instancia (Fluent API)

Método Descripción
setTitle(String) Establece el título.
setMessage(String) Establece el mensaje descriptivo.
setType(Type) INFO, SUCCESS, WARNING o DANGER.
setIcon(JIcon) Icono personalizado (sobreescribe el del tipo).
setConfirmText(String) Texto del botón de confirmación.
setCancelText(String) Texto del botón de cancelar.
setOnConfirm(Runnable) Callback al confirmar.
setOnCancel(Runnable) Callback al cancelar.
setOnConfirmAsync(Supplier, Consumer) Acción asíncrona con loading state.
setShowCancelButton(boolean) Mostrar/ocultar botón cancelar.
show() Muestra el diálogo con animación.
close() Cierra el diálogo con animación.
