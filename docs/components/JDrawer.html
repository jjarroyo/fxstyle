JDrawer

Panel lateral deslizable para contenido contextual, formularios de edición y detalles. Soporta posiciones LEFT/RIGHT, múltiples tamaños, header/body/footer y animaciones slide.

Uso Básico

Un drawer simple con título y contenido.

JDrawer drawer = new JDrawer();
drawer.setTitle("Detalles");
drawer.setBody(new Label("Contenido del drawer"));
drawer.show();

Constructor con Contenido

Pasa el contenido directamente al constructor.

new JDrawer("Mi Drawer", new Label("Contenido")).show();

Posiciones

El drawer puede deslizarse desde la derecha (default) o la izquierda.

Position.RIGHT

Por defecto

Position.LEFT

Menús, filtros

new JDrawer()
    .setTitle("Panel Izquierdo")
    .setBody(content)
    .setPosition(JDrawer.Position.LEFT)
    .show();

Tamaños

SMALL

300px

MEDIUM

420px

LARGE

600px

FULL

100%

new JDrawer()
    .setTitle("Panel Grande")
    .setBody(content)
    .setSize(JDrawer.Size.LARGE)
    .show();

Con Header, Body y Footer

Estructura completa con botones de acción en el footer.

JButton saveBtn = new JButton("Guardar");
saveBtn.addClass("btn-success");

JButton cancelBtn = new JButton("Cancelar");
cancelBtn.addClass("btn-light-danger");

JDrawer drawer = new JDrawer()
    .setTitle("Editar Usuario")
    .setBody(formLayout)
    .setSize(JDrawer.Size.LARGE);

saveBtn.setOnAction(e -> {
    drawer.close();
    guardarDatos();
});
cancelBtn.setOnAction(e -> drawer.close());

drawer.setFooter(cancelBtn, saveBtn);
drawer.show();

Opciones Avanzadas

Deshabitar cierre por backdrop y callback al cerrar.

new JDrawer()
    .setTitle("Formulario Obligatorio")
    .setBody(form)
    .setCloseOnBackdropClick(false)   // Solo cierra con botón X
    .setOnClose(() -> System.out.println("Drawer cerrado"))
    .show();

API Reference

Constructores

Constructor Descripción
JDrawer() Drawer vacío.
JDrawer(Node body) Con contenido directo.
JDrawer(String title, Node body) Con título y contenido.

Métodos (Fluent API)

Método Descripción
setTitle(String) Establece el título del header.
setBody(Node) Establece el contenido del body (scrollable).
setFooter(Node...) Agrega nodos al footer (botones de acción).
setPosition(Position) LEFT o RIGHT (default: RIGHT).
setSize(Size) SMALL, MEDIUM, LARGE o FULL.
setCloseOnBackdropClick(boolean) Habilitar cierre al clic en backdrop (default: true).
setShowHeader(boolean) Mostrar/ocultar header completo.
setOnClose(Runnable) Callback ejecutado al cerrar.
addClass(String...) Agrega clases CSS al panel.
show() Muestra el drawer con animación slide.
close() Cierra el drawer con animación slide.
