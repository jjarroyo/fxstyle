JTreeView

Componente de árbol jerárquico que permite mostrar estructuras de datos complejas de forma visual.

Component

✨ Características

  • Animaciones de despliegue suaves en el ícono de "folder" (chevron rotativo).
  • Integración nativa con JIcon para personalización individual de nodos (ej. carpeta vs archivo).
  • Soporte integrado para casillas de verificación (JCheckBox).
  • API fluida y manejo de eventos fácil mediante listeners.

💻 Ejemplo: Árbol de Archivos

Implementación recomendada utilizando la clase interna JTreeItem para soporte de iconos.

JTreeView<String> treeView = new JTreeView<>();

// Nodo raíz
JTreeItem<String> root = new JTreeItem<>("Proyecto Demo", JIcon.FOLDER);
root.setExpanded(true);

// Hijos
JTreeItem<String> srcFolder = new JTreeItem<>("src", JIcon.FOLDER);
srcFolder.getChildren().addAll(
    new JTreeItem<>("Main.java", JIcon.FILE),
    new JTreeItem<>("utils.java", JIcon.FILE)
);

JTreeItem<String> readme = new JTreeItem<>("README.md", JIcon.FILE_TEXT);

root.getChildren().addAll(srcFolder, readme);
treeView.setRoot(root);

☑️ Ejemplo: Árbol con Checkboxes

Habilitar checkboxes automáticamente enlaza la interfaz de usuario con la propiedad checkedProperty de los JTreeItem.

JTreeView<String> checkTree = new JTreeView<>();
checkTree.setShowCheckboxes(true);

JTreeItem<String> root = new JTreeItem<>("Permisos", JIcon.SETTINGS);
JTreeItem<String> editUser = new JTreeItem<>("Editar Usuario", JIcon.PERSON);

root.getChildren().add(editUser);
checkTree.setRoot(root);

// Escuchar los eventos de checkbox individualmente en todo el árbol
checkTree.setOnCheckAction(item -> {
    System.out.println("Nodo interactuado: " + item.getValue());
    System.out.println("Está seleccionado: " + item.isChecked());
});

⚙️ API Reference (JTreeView)

Método Tipo Descripción
setShowCheckboxes(boolean) void Habilita o deshabilita la visualización de los checkboxes en todo el árbol.
setOnCheckAction(Consumer<JTreeItem<T>>) void Establece un listener invocado al marcar/desmarcar un checkbox. La lambda provee acceso al JTreeItem interactuado.
showCheckboxesProperty() BooleanProperty Propiedad observable de la visibilidad de los checkboxes.

⚙️ API Reference (JTreeItem)

Método Tipo Descripción
setIcon(JIcon) void Asigna un icono SVG desde la enumeración JIcon al nodo para mostrar junto a la etiqueta.
setChecked(boolean) void Establece programáticamente si el checkbox de este nodo está marcado o no.
isChecked() boolean Retorna verdadero si este nodo está marcado.
checkedProperty() BooleanProperty La propiedad observable enlazada directamente al JCheckBox del nodo.

Nota: Hereda de javafx.scene.control.TreeView y javafx.scene.control.TreeItem, por lo que toda la API estándar de JavaFX está disponible (ej. getSelectionModel().getSelectedItem(), setExpanded(true), etc).
